# Natanja Constructora — Demo

Sitio demo de landing page para constructora, desarrollado por **Vonoa Web**.

- HTML/CSS/JS vanilla, sin dependencias ni build.
- Contenido e imágenes de muestra (Unsplash) — se reemplazan con el material real del cliente.
- Hospedado en GitHub Pages.

## Ver en local

Abrir `index.html` en el navegador, o servir con:

```
python -m http.server 3004
```
